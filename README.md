Restfull service для Московского Авиационного Института. 
=========================================================


**Запуск приложения:**
-------------------
**1. Установить переменную среды FLASK_APP:**
* set FLASK_APP=restfull.py (для Windows)
* export FLASK_APP=restfull.py (для Linux)  
**2. Выполнить команду:**  
* flask run

Стек технологий: 
----------------
* Python
* Flask 
* SQLAlchemy
